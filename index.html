CSS Units

%

Relative to the value of the parent element. 100% width is the full width of the parent element. Not relative to the element itself!

em

Relative to font-size of the parent element. If the value is 1.1em that's the same as saying 110% of the parent's font size.

vh

Units are a percent of the viewport's height. So 1vh is equal to 1% of the window height, while 100vh is the full height of the window.

px

Absolute size in pixels. It's important to know that CSS pixels are not physical, but merely references, so a 50x50 pixel box will be roughly the same size on a low-density screen as it is on a high-density screen.

rem

Relative to the font size of the root (i.e. <html>) element. So a size of 1.5rem is equal to 150% of the root font size.

vw

Units are a percent of the viewport's width. So 1vw is equal to 1% of the window width, while 100vw is the full width of the window.
